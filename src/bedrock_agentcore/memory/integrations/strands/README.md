# Strands AgentCore Memory integrations

Choose the guide for the Strands memory integration you want to use:

- [Session manager](memorysessionmanager/README.md) — persist Strands sessions in AgentCore Memory and restore short-term conversation state, with optional long-term memory retrieval.
- [Memory store](memorystore/README.md) — connect AgentCore long-term memory to Strands' native `MemoryManager` for recall and extraction.
